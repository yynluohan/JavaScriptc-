# moment常用方法 #

1. 获取当前时间

```js
moment().format()
//格式如 2020-01-20T14:53:36+08:00
```

2. 获取今天开始~结束时间

```js
moment().startOf('d').format()  // 2020-01-20T00:00:00+08:00
moment().endOf('d').format() // 2020-01-20T23:59:59+08:00
```

3. 获取当月开始~结束时间

```js
moment().startOf('month').format()  // 2020-01-01T00:00:00+08:00
moment().endOf('month').format()  // 2020-01-31T23:59:59+08:00
```

4. 时间加减

```js
// 加一年
moment().add(1, 'year').format()  //2021-01-20T15:06:19+08:00

// 加一个月
moment().add(1, 'month').format()  //2020-02-20T15:06:19+08:00

// 加1天
moment().add(1, 'day').format()  // 2020-01-21T15:06:19+08:00

// 加1小时
moment().add(1, 'hour').format()  // 2020-01-21T16:06:19+08:00


// 固定时间加天数
moment('2020-01-20T16:09:47+08:00').add(1, 'day').format()  // 2020-01-21T16:09:47+08:00
moment('2020-01-20').add(1, 'day').format())  // 2020-01-21T00:00:00+08:00
```

5. 转时间戳(可根据时间戳比较时间大小)

```js
moment('2020-01-20T16:09:47+08:00').unix()  // 1579507787
moment('2020-01-20').add(1, 'day').unix() // 1579536000
```
